# ♟️ Spell Chess

A fun twist on classic chess where you can use powerful spells to outplay your opponent.

---

## 🎮 Features

- Play classic chess with magical abilities
- Use special spells like:
  - ❄️ Freeze opponent pieces
  - ⚡ Double move
  - ✨ Unique special abilities
- Player vs Player mode
- Undo / Redo moves
- Smooth animations and simple controls

---

## 🕹️ Controls

| Action        | Input               |
|---------------|---------------------|
| Select Piece  | Left Click          |
| Move Piece    | Left Click          |
| Use Spell     | Click spell button  |
| Undo          | Z                   |
| Redo          | Y                   |
| Restart       | R                   |
| Quit          | Q / ESC             |
| Resign        | Click resign button |
